Detailed Description

template<class iter_t>
class hpx::parallel::sort::tools::buffer_guard< iter_t >

This is a stack of bufferes initialized for to be used by the threads The idea is like the lock_guard, you create the object and obtain an iterator to a buffer, and when the object is destroyed, the buffer is automatically returned to the stack.
